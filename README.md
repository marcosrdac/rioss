# RIOSS

**Radar Image Oil Spill Seeker (RIOSS)**, is an open-source machine learning classification tool to help identifying oil spills on ocean surface from radar satellite imagery. The project RIOSS, coordinated by Luis Felipe de Mendonça, was first financed by CNPq. As of today, it contemplates a part of the broader project BR-Oil by INCT-GP.
